# Weather App

## Overview
This Weather App provides real-time weather information for various locations. It offers a user-friendly interface to easily access weather forecasts and current conditions.

## Features
- **Current Weather:** Get the current weather conditions including temperature, humidity, wind speed, and more for a specific location.
- **Responsive Design:** Access the app seamlessly on various devices - desktops, tablets, and mobile phones.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** OpenWeatherMap API (for weather data retrieval)

## Installation
1. Navigate to the project directory.
2. Open `index.html` in your preferred web browser.
